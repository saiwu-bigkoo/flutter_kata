## 简单用例
#### 1.在 pubspec.yaml 中添加依赖
```
//pub方式
dependencies:
  flutter_kata: ^1.0.0

//导入方式
dependencies:
  flutter_kata:
    path: 项目路径

//git方式
dependencies:
  flutter_kata:
    git:
      url: git://github.com/saiwu-bigkoo/flutter_kata.git
```
